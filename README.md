# Tibetan Font Toolkit

A toolkit focused on solving common Tibetan script typesetting and rendering issues.

## Project Goal

This project aims to address various display and layout problems that Tibetan text frequently encounters in digital environments, such as:

- Stacked character display issues (上下叠字)
- Left-right structural glyph problems (左右指)
- Complex Tibetan character rendering bugs across platforms
- Compatibility issues with mainstream typesetting software (Microsoft Word, Adobe InDesign, Google Docs, etc.)
- Font rendering optimization and testing tools

## Current Status

- Documentation of common issues
- Planning practical scripts and tools

## 中文介绍

# 藏文字体工具包

一个专注于解决藏文排版和渲染问题的开源工具集。

## 项目目标

本项目致力于解决藏文在数字环境中常见的显示与排版问题，主要包括：

- 上下叠字（叠加字符）显示异常
- 左右结构指（左右指）排版错误
- 复杂藏文结构在不同平台下的渲染问题
- 与主流排版软件（Microsoft Word、Adobe InDesign、Google Docs 等）的兼容协调问题
- 字体渲染优化与测试工具

## 后续计划

- 完善更多实用脚本和工具
- 撰写详细的使用文档和示例
- 收集并解决社区中真实的藏文排版问题

欢迎藏文使用者、开发者提交 Issue 和 Pull Request，一起完善这个工具包！
