# Changelog

## v0.1.0 - Early Preview

Initial public early-preview release.

### Added

- Added `scripts/diagnose_font.py` for read-only font diagnostics.
- Added `scripts/normalize_tibetan.py` for Tibetan Unicode text normalization.
- Added `scripts/generate_test_pages.py` for generating HTML rendering test pages.
- Added sample Tibetan text and rendering documentation in `examples/`.
- Added pytest tests for normalization behavior.
- Added GitHub Actions workflow for automated test runs.
- Added MIT License.
- Added early-preview safety notes in README.

### Notes

- This release is intended for testing, research, documentation, and community feedback.
- Production use, bulk font processing, and important document workflows still require manual verification.
